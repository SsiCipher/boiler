# Boiler - auto-start projects

a simple bash script to automate the process of starting a project by creating the initial boilerplate.


## Installation


```bash
$> git clone https://github.com/SsiCipher/boiler.git
```

```bash
$> chmod 755 installer.sh 
```

```bash
$> . ./installer.sh
```

## Usage

```bash
$> boiler [Project Name]
```
